# Requisitos Funcionais

## RF01

O sistema deve permitir o cadastro de empresa do tipo Pessoa Jurídica.

## RF02

O sistema deve permitir o cadastro de empresa do tipo Pessoa Física.

## RF03

O sistema deve permitir o cadastro de empresa do tipo Pessoa Estrangeira.

## RF04

O sistema deve permitir informar o perfil da empresa.

## RF05

O sistema deve permitir anexar documento comprobatório obrigatório.

## RF06

O sistema deve validar duplicidade entre documento obrigatório e documento opcional, quando houver documento opcional.

## RF07

O sistema deve permitir habilitar faturamento direto.

## RF08

O sistema deve validar CPF informado.

## RF09

O sistema deve validar CNPJ informado.

## RF10

O sistema deve validar os campos obrigatórios.

## RF11

O sistema deve exibir mensagem de sucesso após cadastro.

## RF12

O sistema deve permitir que usuários internos gerenciem o status do cadastro da empresa.

## RF13

O sistema deve permitir que usuários internos aprovem empresas cadastradas.

## RF14

O sistema deve permitir que usuários internos reprovem empresas cadastradas.

## RF15

O sistema deve permitir vincular um usuário externo como responsável pela empresa.

## RF16

O sistema deve redirecionar o usuário após a conclusão do cadastro, conforme o fluxo executado.

## RF17

O sistema deve permitir o cadastro de empresas por usuários internos.

## RF18

O sistema deve permitir o upload de arquivos nos formatos PDF, PNG, JPG e JPEG.

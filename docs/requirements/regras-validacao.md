# Regras de Validação

## RV01

Os campos obrigatórios deverão respeitar a quantidade mínima de caracteres definida pelo sistema.

## RV02

O CNPJ deverá possuir 14 caracteres válidos.

## RV03

O CPF deverá possuir 11 caracteres válidos.

## RV04

O perfil da empresa deverá ser obrigatoriamente informado.

## RV05

O documento comprobatório obrigatório deverá ser anexado antes da conclusão do cadastro.

## RV06

Somente arquivos nos formatos PDF, PNG, JPG e JPEG poderão ser anexados.

## RV07

Não será permitido anexar arquivos duplicados entre os documentos obrigatórios e opcionais.

## RV08

O sistema deverá exibir mensagem de alerta quando um campo obrigatório não atender ao tamanho mínimo permitido.

## RV09

O sistema deverá exibir mensagem de alerta quando o CPF informado for inválido.

## RV10

O sistema deverá exibir mensagem de alerta quando o CNPJ informado for inválido.

## RV11

O sistema deverá exibir mensagem de alerta quando o perfil não for informado.

## RV12

O sistema deverá exibir mensagem de alerta quando o documento obrigatório não for anexado.

## RV13

O sistema deverá exibir mensagem de alerta quando o formato do arquivo for inválido.

## RV14

O sistema deverá exibir mensagem de alerta quando houver tentativa de anexar arquivos duplicados.
